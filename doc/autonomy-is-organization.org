* Vision
 
 + It's all *text*
 + It makes you *autonomous*
 + It fosters *data resiliency*
 + It waves patterns for *organic growth* of information

* Purpose

Write a guide and a toolkit implementing [[http://www.davidco.com/][Getting Things Done]] (GTD) via
traditional GNU tools and languages, primarily [[http://www.gnu.org/software/emacs/][Emacs]], [[http://orgmode.org][org-mode]], [[http://git-scm.com/][Git]]
and portable Shell scripts, integrating them into the Desktop.

* Goals

  - a plain text authoring platform

  - a personal information system

  - a publishing system

  - an authenticated, federated and revisioned framework for documents

  - a book describing GTD workflow in AutOrg
